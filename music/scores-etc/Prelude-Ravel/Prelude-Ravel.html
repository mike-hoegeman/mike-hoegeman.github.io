Prelude (Ravel 1913)

This page is a guide to learning Prelude (Ravel 1913) on a tapping instrument such as the Chapman Stick or the acoustic Dragonfly DFA.

There is a score/tablature that will help you along the way. These tabs/fingerings are for a 12 string matched reciprocal tuning.

Musescore score + tabs

This is the Authoritative version which is for twelve string Matched reciprocal tuning.

There may be others in the future with different tunings. If you have a request for a different tuning let me know at mike.hoegeman@gmail.com

This transcription referred to one done by Richard Yates for solo guitar.

Background

  • This piece is by Maurice Ravel
  • Anders Miolin does a nice version of this on a classical 11 string Alto guitar .

Elemental Structure

This version of the piece is transposed to E minor from the original A minor for piano.
